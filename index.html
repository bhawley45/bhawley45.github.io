Previous Adventures!

Simple Solar System

A pretty basic modeled Solar System. The main camera isn't movable but each planet rotates and orbits at it's actualSpeed / Scalar. Learned how to use different skyboxes and light emmision in Unity.

Castle Conquest!

First fully developed gameplay loop. Also my first experience with 2D Unity and pixelart. In the game you play as a prince who's kingdom was taken over by pigs, you have to beat them all and reclaim the castle... Had a lot of ideas for this project but had to cut some corners to make it managable.

Droppin' Tilted

This is my first from-scratch game. It's a puzzle game where you take control of a platform to guide a ball through a fun little maze. This is a fairly rough project where adaptive planning was necessary but I'm still proud of it. I have many more levels outlined, but curently only 1 is functional with lots of areas to improve. Enjoy :)

Crossbow Training

This project was intense and required a lot more learning and adapting than I expected. This is my first project in Unreal Engine and is a small FPS game where you try and shoot all the targets before your time runs out. Different scores are assigned based on the target and location hit.
